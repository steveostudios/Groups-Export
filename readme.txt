== Description ==

Easy CSV (Comma Separated Value) export of user data, sorted by [itthinx Groups plugin](http://www.itthinx.com/plugins/groups/). Data includes User ID, Email, First Name, Last Name, and First and Last Name. 

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. Go to Settings -> Groups Export in the admin sidebar to export stuff

== Frequently Asked Questions ==

= I installed the plugin... Where is it? =

Go to Settings -> Groups Export in the Wordpress admin sidebar

= I clicked "Download CSV"... What happened? =

Check your download location.  

= Where is my Download location? =

Well, this can vary depending on your OS, browser, and user settings. For me (Mac OSX, Safari, defaults) it is my "Downloads" folder.
